# SwitchRoms Source

Sorgente per scaricare ROM Nintendo Switch da [SwitchRoms.io](https://switchroms.io/).

## Caratteristiche

- ✅ Ricerca ROM per titolo
- ✅ Download multipli (NSP, XCI, Update)
- ✅ Supporto WebView per link di download esterni
- ✅ Solo piattaforma Nintendo Switch

## Note

- I link di download richiedono l'apertura di un WebView per ottenere l'URL finale
- Le regioni non sono disponibili su SwitchRoms
